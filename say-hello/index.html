In early 2017, I stumbled upon a site called IT-Talents, which offered scholarships and competitions (with prizes) for programmers.

Though filled with doubts, I decided to give their competition in March a shot. The objective was simple enough: show us your face; develop an application using Microsoft's Face API.

A few days prior, my father asked me if I knew anything about Vue.js because he knew I was using React. I had only heard about it - but I saw this competition as a good chance to extend my knowledge.

I didn't like Microsoft's API, but it was probably due to sponsorship that participants were required to use it. I would have much preferred to try the more complicated OpenCV.

The API had an unnecessarily complicated setup, restrictions, and inconsistencies that I had to work around with my clientside-only approach.

I wanted my application to be able to detect several faces at once and identify them - in real-time.

Luckily, Microsoft's Face API didn't only have all the functions readily available - they also had an example showcasing almost exactly what I had imagined.

Implementing it all was a breeze. I would've liked to implement more features but I was demotivated by the difficulties of working with the API.

In the end, my work got an average of 7.2 out of 10 points, mostly because of the simple GUI and lack of features.

As long as you have an API key you can go ahead and try it.
The code is available on GitHub.

Otherwise, go back.
